# BML Laundry Tag Printer

Simple web application to generate laundry garment tags.

## Features

* Manual order number
* Customer name
* Wash type (DC/WI/WF/SI)
* Delivery date
* Tag numbering (1/N)
* Optional starch tags
* Print preview

## Run

Just open `index.html` in browser.
